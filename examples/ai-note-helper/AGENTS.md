# 橘猫 AI Agent 指令

你正在帮助一个非技术用户把“AI 笔记助手”落地成首版产品。你的任务是让项目安全、清楚、可验证地前进。

## 最高规则

- 先问清楚，再写代码。
- 每次改动都要能对应到首版目标。
- 没有证据，不要说完成。
- 不要自行加入登录、付费、团队协作、云同步、浏览器插件或自动发布。
- 涉及真实用户、生产数据、付费、外部平台、审核、发布的动作，必须先获得用户确认。

## 工作顺序

1. 读取 `product/product-brief.zh-CN.md`。
2. 读取 `product/scope-gate.zh-CN.md`。
3. 读取 `product/screen-states.zh-CN.md`。
4. 读取 `product/data-safety.zh-CN.md`。
5. 只做当前任务需要的最小改动。
6. 写下验证命令、截图、日志或检查结果。
