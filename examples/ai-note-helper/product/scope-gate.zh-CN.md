# 首版边界：AI 笔记助手

## 首版必须做

- 新建一条文本笔记。
- 点击“整理”后生成标题、摘要和三条下一步行动。
- 展示整理结果。
- 允许用户复制整理结果。
- 本地保存最近的笔记和整理结果，方便返回查看。
- 整理失败时，告诉用户发生了什么，并保留原文。

## 首版明确不做

- 不做登录和账号体系。
- 不做团队协作。
- 不做付费、订阅、试用转订阅。
- 不做浏览器插件。
- 不做云同步。
- 不自动发送邮件、消息或发布内容。
- 不做多模型选择器。

## 以后再做

- 多设备同步。
- 标签和搜索。
- 导出 Markdown 或 PDF。
- 用户自带 API Key 的设置页。
- 团队共享。

## 不要让 AI 自己加

- 不准加登录。
- 不准接入真实支付。
- 不准上传用户真实笔记到未确认的第三方服务。
- 不准把“整理结果”改成自动执行任务。
- 不准把这个练习工作区说成已经上线的真实产品。

## 需要人工确认的动作

- 涉及真实用户。
- 涉及生产数据。
- 涉及付费。
- 涉及上线或审核。
- 涉及外部平台账号。
- 涉及把真实笔记发给任何模型服务。
