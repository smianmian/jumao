# First-Version Scope: AI Note Helper

## Must have

- Create a text note.
- Generate a title, summary, and three next actions after the user clicks "Organize".
- Display the organized result.
- Let the user copy the organized result.
- Store recent notes and results locally so the user can return to them.
- If organization fails, explain what happened and keep the original note.

## Explicitly out of scope

- Login and account system.
- Team collaboration.
- Payments, subscriptions, or trial-to-paid flows.
- Browser extension.
- Cloud sync.
- Automatic email, messaging, or publishing.
- Multi-model picker.

## Later

- Multi-device sync.
- Tags and search.
- Markdown or PDF export.
- Settings page for a user-provided API key.
- Team sharing.

## Do not let AI add by itself

- Do not add login.
- Do not connect real payments.
- Do not upload real user notes to an unconfirmed third-party service.
- Do not turn organized results into automatic task execution.
- Do not claim this example is a launched real product.

## Actions that need human confirmation

- Real users.
- Production data.
- Payments.
- Launch or review.
- External platform accounts.
- Sending real notes to any model service.
